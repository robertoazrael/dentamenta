# DentaMenta Marketing Site

Este proyecto es una migración del prototipo original a **Astro + Tailwind CSS**.

## Tecnologías
- **Astro**: Framework para sitios web rápidos.
- **Tailwind CSS**: Estilizado mediante utilidades.
- **Lucide Astro**: Iconos vectoriales optimizados.
- **Vanilla JS**: Para comportamientos interactivos mínimos.

## Inicio Rápido

Para ejecutar el proyecto localmente, asegúrate de tener [Node.js](https://nodejs.org/) instalado y sigue estos pasos:

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Iniciar servidor de desarrollo:**
   ```bash
   npm run dev
   ```

3. **Construir para producción:**
   ```bash
   npm run build
   ```

## Estructura del Proyecto
- `src/components/`: Secciones individuales del sitio (Hero, Servicios, etc.).
- `src/layouts/`: Plantilla base (Layout.astro).
- `src/pages/`: Páginas del sitio (index.astro).
- `src/styles/`: Estilos globales y configuraciones de Tailwind.
